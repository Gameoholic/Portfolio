Partigon

Showcase of particle animations made with Partigon

This project is open source! View its repository at the bottom of this page.

Partigon is a library which makes it easier for plugin developers to create complex particle animations. It's written in Kotlin, for use in Paper plugins.
Partigon is a powerful library with a lot of different concepts, I'd recommend taking a look at the documentation.

In short, particles are created through either of the PartigonParticle interface implementations (SingularParticle or MultiParticle).
All properties are controlled via Envelopes. An Envelope is essentially an interpolation of a value over time. There are different types of Envelopes, and Envelope Wrappers, which ease the process of creating more complex shapes. (curves, circles..)
Envelopes require Loops. Loops determine the frame index passed to the envelope, and add support for repeating and reversing animations.

Different Loop types

It's also possible to pass rotation options to envelopes & particles, which results in nice animations when used creatively

Because Partigon is written in Kotlin, it takes advantage of Kotlin concepts, such as the DSL builder pattern, inline methods & properties, and extensions. It makes the library much more boilerplate-free and nicer to work with.
