Twitch Rewards

The video has sound!

Twitch Rewards is a Spigot plugin that connects to streamers' chats using the twitch4j Twitch API client.
After linking to the streamer's chat, they can customize what happens when redeems are activated on their stream.
Some examples are:

  • Spawn a creeper
  • Give a random potion effect
  • Lock an inventory slot
  • Spawn an air drop
  • Activate TNT run (blocks crumble below you)
  • Activate a clutch challenge

The 4 different airdrop rarities

The plugin is customizable through the config.yml file, and every redeem type has options which can be configured to the streamer's liking. It can also affect multiple streamers from different chats.
